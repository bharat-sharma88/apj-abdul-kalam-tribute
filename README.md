Tribute Website – Dr. A.P.J. Abdul Kalam:-

This is a tribute website dedicated to Dr. A.P.J. Abdul Kalam, the Missile Man of India and the 11th President of India.

🚀 Features:-

Clean and modern UI

Responsive design

Hero section with circular image

About section

Achievements cards

Inspirational quotes

Footer design with gradient effects

🛠 Built With:-

HTML5

CSS3

Flexbox

Google Fonts (Poppins)

Font Awesome Icons

📷 Sections Included:-

Hero Section

About Dr. Kalam

Achievements

Inspirational Quote

Footer

📌 Purpose:-

This project was built as a frontend practice project to improve HTML & CSS skills.
